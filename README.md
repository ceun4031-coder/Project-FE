#StoryLex 

StoryLex는 **오답 단어를 AI 스토리로 자연스럽게 반복 노출**시켜, 단어를 “외우는 것”이 아니라 **문맥으로 익히게 만드는 영어 학습 서비스**입니다.
퀴즈/플래시카드 학습으로 오답을 모으고, 오답노트에서 복습하거나 **AI 스토리로 다시 만나며** 학습 성과(출석/주간 그래프/Top5)를 대시보드에서 확인할 수 있습니다.

---

## Tech Stack
- **Vite 7**: 번들러/개발 서버
- **React 19**: UI 프레임워크
- **React Router DOM 7**: 라우팅
- **TanStack Query v5**: 서버 상태/캐싱/데이터 패칭
- **Zustand v5**: 클라이언트 전역 상태(Auth, 학습 설정)
- **Axios**: HTTP 클라이언트
- **Recharts**: 대시보드 차트
- **Lucide React / Font Awesome**: 아이콘

---

## Service Flow
- **비회원**: 메인(소개/가이드)만 이용
- **회원**
  1. 가입/로그인
  2. **대시보드**에서 오늘 상태 확인
  3. **단어장 / 학습하기**에서 학습 진행
  4. 결과에서 **오답 단어 누적**
  5. **오답노트**에서 재학습 또는 **AI 스토리**로 문맥 반복
  6. 대시보드에서 성과(출석/그래프/Top5) 확인
  7. 다시 학습 진행(반복)

## Features (회원 기능 상세)
### 1) Dashboard (홈)
**목적:** 오늘 상태 확인 + 다음 행동 유도
- 오늘 학습량 / 목표 대비 달성률
- 누적 학습 완료 단어 수
- 연속 학습 일 수(스트릭)
- 1주 출석 현황
- 주간 학습 활동량(그래프)
  - 카드/퀴즈 정답/오답을 **막대 그래프**로 시각화
- 자주 틀리는 단어 TOP 5
  - 기준: 오답 횟수
- 바로가기
  - 학습하기(정규 퀴즈) / 복습하기(오답 퀴즈)

---

### 2) Word List (단어장)
**목적:** 단어 탐색 → 단어 상세 → 학습/즐겨찾기

#### (1) 목록 화면
- 탭
  - 전체 단어(개수 표시)
  - 즐겨찾기(개수 표시)
- 필터/검색
  - 품사 드롭다운
  - 분야 드롭다운
  - 난이도 드롭다운
  - 검색
- 단어 카드 구성
  - 난이도 / 품사 / 분야
  - 영어 단어 / 한국어 뜻
  - 즐겨찾기(추가/취소)
  - 단어 상세 이동

#### (2) 단어 상세
- 기본 정보
  - 난이도 / 품사 / 분야
  - 학습 상태 태그: **학습 예정 / 학습 완료 / 재학습 예정**
  - 예문
  - 영어 뜻 / 한국어 뜻
  - 즐겨찾기(추가/취소)
- 연관 단어(클러스터)
  - 유의어 / 반의어
  - 단어 클릭 시 해당 단어 상세로 이동
- 학습 통계
  - 퀴즈/카드 정답 횟수, 오답 횟수
  - 정답률(계산)
  - 마지막 학습일
  - 바로 학습(단어 1개 전용)
    - 퀴즈 1문항 시작
    - 카드 1개 시작

---

### 3) Learning (학습하기)
**목적:** 조건 선택 → 학습 진행 → 결과 확인

#### (1) 학습 설정
- 문항 수 선택
- 난이도 선택
- 분야 선택
- 모드 선택
  - 정규 학습
  - 오답 학습(오답 단어 기반)

#### (2) 학습 모드
**A. 퀴즈 학습**
- 객관식 4지선다(1개 선택)
- 모드: 정규 / 오답

**B. 플래시 카드 학습**
- 카드 클릭으로 뒤집기(뜻 확인)
- 아는지/모르는지 선택
- 모드: 정규 / 오답

#### (3) 학습 결과
- 요약
  - 정답 수 / 오답(또는 모름) 수
  - 정답률
- 시각화
  - 정답/오답 수 바 형태 표시
- 목록
  - 틀린/모르는 단어 리스트 제공
- 다음 행동
  - AI 스토리 생성(오답 기반) 유도

---

### 4) Wrong Note (오답노트)
**목적:** 틀린 단어 관리 → 재학습/스토리로 확장

#### (1) 오답 목록
- 정렬
  - 최신순 / 오래된순 / 많이 틀린순
- 단어 선택(복수)
- 상태
  - 스토리 사용 여부(미사용/사용) 구분 가능

#### (2) 오답 액션
- 오답 학습 시작
  - 선택한 단어들로 퀴즈/카드 학습
- 오답 스토리 생성
  - 조건: 스토리에 **미사용 오답 단어**만 선택
  - 흐름:
    - 오답노트에서 단어 선택 → 스토리 생성 페이지 이동
    - 이동 시 선택 단어 **자동 체크 상태로 반영**

---

### 5) AI Story (AI 스토리)
**목적:** 오답 단어 반복 노출 + 맥락 학습

#### (1) 스토리 목록
- 오답 단어 기반 스토리 리스트
- 검색 기능

#### (2) 스토리 상세
- 좌측: 사용된 단어 목록
  - 단어 호버 시 우측 본문에서 해당 단어 강조
- 우측: 영문 소설
  - 제목 / 내용
  - 하단: 한국어 번역(제목/내용)
- 기능
  - 스토리 삭제
